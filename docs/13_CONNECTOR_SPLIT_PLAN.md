# Connector Split Plan

For marketplace trust, keep high-permission connectors modular.

## Initial package

May include connector scaffolds and local implementations.

## Future packages

- Core Artist Runtime.
- Suno connector.
- X/Bird connector.
- Instagram connector.
- TikTok connector.

## Why split

- Users can install only what they need.
- Security review is easier.
- OAuth setup docs are platform-specific.
- Platform API churn does not destabilize the core artist runtime.
