# Social Connectors Spec

Platforms have different roles.

## X

Role: public voice.
Connector: Bird.

Default auto post types:

- observations;
- studio notes;
- lyric fragments;
- demo teasers;
- new song links.

Replies and quotes should be separately authorized.

## Instagram

Role: visual identity.
Connector: Instagram Content Publishing where available.

Default auto post types:

- lyric cards;
- cover visuals;
- Reel teasers.

## TikTok

Role: discovery and short video.
Connector: TikTok Content Posting where available.

Default auto post types:

- hook clips;
- demo teasers.

## Connector contract

All connectors must expose capabilities, execute through authority guard, and return audit-safe results.
