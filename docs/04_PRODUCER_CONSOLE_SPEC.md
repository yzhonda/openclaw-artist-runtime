# Producer Console Spec

Producer Console is the control tower, not the daily workflow.

## Pages

1. Dashboard — status, autopilot state, recent actions, alerts.
2. Setup — account connections and platform selection.
3. Music / Suno — browser worker state, budget, ledgers, runs.
4. Platforms — X, Instagram, TikTok roles and authority.
5. Songs — active works, prompt packs, takes, assets.
6. Prompt Ledger — inspect prompt lineage.
7. Audit Log — inspect side effects and failures.
8. Settings — budgets, cadence, hard stops, dry-run.
9. Recovery — reconnect, pause, clear lock, export diagnostics.

## UI principles

- Speak producer language, not implementation language.
- Show what the artist is doing and why.
- Default to status and recovery, not manual micromanagement.
- Every connected platform shows capabilities and hard-stop state.
