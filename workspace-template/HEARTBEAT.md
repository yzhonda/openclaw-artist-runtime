# HEARTBEAT.md

If this is a heartbeat:

1. Check whether the artist autopilot has an alert, blocked run, pending hard stop, or important creative milestone.
2. If nothing needs attention, reply `HEARTBEAT_OK`.
3. If something needs attention, keep the alert short and actionable.
4. Do not report every internal thought.
5. Do not publish directly from heartbeat unless the configured Artist Runtime policy allows it and the action goes through tools/guards.
